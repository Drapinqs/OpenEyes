Tools and Resources

Digital Breadcrumbs/Open Source Intelligence - OSINTFramework, OSINTTechniques
Digital Breadcrumbs are used to pick up the little traces that people leave behind on the internet. These can be useful for finding missing loved ones, or researching people and doing background checks if you are in the position for hiring them.
(URL) Link Shorteners - V.HT, TinyURL, Bit.ly, u.nu
URL Shorteners are used to redirect you to a large link. For example, you can change an extremely long link to a .pdf file, into just a few characters.
Windows 10/11 CMD Commands - SS64
List of commands that are used in the Windows 10 and Windows 11 command prompt. Mostly used by IT specialists, typically for computer repair.
Check Information for Data Breach Leaks - HaveIBeenPwned, Passwords, DEHASHED
Search if your passwords, emails, or phone numbers have been leaked on the internet.
Remove Advertisements Extension - uBlockOrigin(Chrome), uBlockOrigin(Edge), uBlockOrigin(Firefox)
Remove any unwanted advertisements for modern-day browsers by adding this open-source extension.
YouTube Video/Audio Downloader - yt2mp3, yt5s, y2mate
Extract the video or audio from a YouTube video online. Make sure to check the copyright, or have the owner's permission before using the video.
Online HTML Editor - W3Schools, CodePen, JSFiddle
Create or edit website pages by using one of these applications.
Operating Systems - Windows10, Windows11, Kali, Ubuntu, Zorin
Operating systems manage all of the software configured on your devices.
Programming Languages/IDE's - Python, Java, VisualStudio, WebStorm, PyCharm
These applications make editing code much easier, and more productive, by providing you with a full code editing environment.
Online Text Hosting - Hastebin, Ghostbin
Upload any text message to these servers, where only people who have the link can read them.
Technology News - TheHackerNews, Wired, BBC
Obtain the latest news in the technological realm.
Free File Hosting - AnonFiles, FileDropper, ufile
Similar to Text Hosting, you can host files online to share with anybody you give the link to.
File Management - 7Zip, WinRAR
Manage your folders and files on your computer. You can upload any of your files by using one of the links in the Free File Hosting section.
Text Editors - Notepad++, SublimeText, STEdit
Similar to IDE's or Online HTML Editors, you can use these programs to view any text file, or modify it.
Download Management - FreeDownloadManager, IDM, JDownloader
Manage your downloads on your computer. You can also utilize these programs to improve your download spped by downloading in chunks.
Photo Manipulation - Paint.net, Krita, GIMP
These programs can be utilized to edit any digital picture, or create one.
Additional Web Browsers - Brave, Chromium, Ferny, Puffin
Getting tired of Google Chrome or Edge? Use some of these!
Anonymity - TOR, VPNs, GeoNode, DuckDuckGo, NameGenerator, PrivacyTools
Your information is somewhere out there on the web. These tools are to help keep you as private and anonymous as possible.
Passwords - PasswordStrengthCheck, PasswordGenerator, LastPass
These tools can help you manage your passwords, or even create them.
Currency - Privacy(CreditCardCreator), PayPal, XE
These sites are intended for managing money, or checking the currency exchange rates.
Crypto Currency - CoinMarketCap(CreditCardCreator), Coinbase, Paxful
These sites are intended for checking the crypto-currency exchange rates.
Disposable Emails - YopMail, Temp-Mail, Mailproof
Here you can create emails, that you most likely will never use again. These are great for singing up on sketchy websites.
Windows Optimization - WindowsBoostPack
Make your Windows 10/Windows 11 computer faster!
